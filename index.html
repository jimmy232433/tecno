Come mandare le dispense

Per mandare le dispense basta caricare il file e mettere nella discalia @admin e il tema del file

  • La dispensa per questioni di comodità deve essere in pdf
  • La dispensa deve utilizzare un linguaggio chiaro e basilare
  • Non deve essere più grande di 25 pagine
  • Non può dare falsi insegnamenti

Nel caso in cui la dispensa che hai deciso di condividere è stata pubblicata o è stata rifiutata vi notificheremo all'interno del gruppo

Aiutare le persone all'interno del gruppo

Naturalmente lo scopo principale del gruppo è aiutare le persone nel capire al meglio il mondo dell'informatica quindi all'interno del gruppo è possibile aiutare un persona solo se si ha chiaro l'argomento che si tratta quindi è vietato dare false informazioni o informazioni poco chiare che portano chi chiede aiuto a non capire o a capire in modo errato un determinato argomento.

Come aiutare il gruppo

Ecco alcuni modi per aiutare il gruppo a diventare migliore

  • Invitare i propri amici all'interno del gruppo
  • Inviare più dispense possibili
  • Aiutare il più possibile le persone all'interno del gruppo

Se seguite assiduamente tutti questi step chi lo dice ... Magari potrete anche fare parte dello staff del gruppo

Mandare esempi di codici

Per aiutare la comprensione di un argomento per quanto riguarda un linguaggio di programmazione o non, è possibile mandare degli esempi di codici.

  • Puoi mandarli sotto forma di messaggio nel gruppo in monospace
  • Puoi mandare il link di Github
  • Puoi mandare direttamente il file del codice in formato .txt
